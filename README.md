# AI and Robotics – Object Detection and Bervage Classification Robot


Developed a fully autonomous robot with car look mixed gripper, dynamic turning, real-time object detection and scanning routines using YOLO, Python, VNC, SSH


Sending control frames from a Linux/Windows computer to an STM32 over a USART link


Control the robot to classify different drinks placed in assigned zone by itself

## Cola-Classification.py

The program can classify regular mini cola cans and mini cola (No sugar). 

It will take all the setting number of regular cola cans to the setting zone.

## Cans-Cartons-Classification.py

It can classify cans and cartons and placed them in cans zone and cartons zone.
